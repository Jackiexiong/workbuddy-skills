# 试卷生成（开发中）

> ⚠️ 本技能为占位文件，待开发实现。

## 规划功能

- 按学科 / 学段 / 知识点生成试卷
- 题型模板：选择、填空、判断、解答题
- 难度分级与分值自定义
- 题干与答案分离输出（便于排版与阅卷）
- 支持 Word / Markdown 双格式导出

## 使用场景

- 单元测试、期中 / 期末试卷组卷
- 同知识点多套平行卷生成
