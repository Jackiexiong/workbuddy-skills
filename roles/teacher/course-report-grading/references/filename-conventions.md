# 学生报告文件名解析规范

## 强制命名格式

学生提交的实验报告 / 课程设计报告必须按以下格式命名，否则系统拒绝提交：

```
班级_姓名_学号.docx
```

### 字段定义

| 字段 | 说明 | 示例 |
|------|------|------|
| 班级 | 专业简称 + 年级班号 | `软件2301`、`计科2202`、`数据科学2301` |
| 姓名 | 学生中文姓名 | `陈小鹏` |
| 学号 | 学校学号（数字串） | `120231080315` |

### 完整示例

```
软件2301_陈小鹏_120231080315.docx
计科2202_李四_120221080208.docx
数据科学2301_王五_120231080420.docx
```

## 解析逻辑

按 `_`（下划线）分隔，三段式：

```
parts = filename_without_ext.split("_")
# parts[0] = 班级
# parts[1] = 姓名
# parts[2] = 学号
```

### 校验规则

1. **扩展名**：必须为 `.docx`（小写），拒绝 `.pdf`、`.doc` 等
2. **段数**：必须恰好 3 段，多段或少段均拒绝
3. **班级**：非空，通常含数字（如 `软件2301`）
4. **姓名**：非空，中文字符
5. **学号**：纯数字，长度通常 10–15 位

### 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 文件名不含下划线 | 标记为异常，跳过评分并记入异常清单 |
| 下划线超过 2 个 | 取前 3 段，多余段拼回姓名（容错） |
| 学号含非数字字符 | 标记异常 |
| 扩展名非 .docx | 拒绝并提示学生重新提交 |
| 文件名含特殊字符 | 去除首尾空格后重新校验 |

## 文件名到成绩表的映射

解析后字段直接写入成绩汇总 Excel：

```
文件名：软件2301_陈小鹏_120231080315.docx
  ↓
Excel 行：
  班级=软件2301  姓名=陈小鹏  学号=120231080315
```

> 评语中以"班级-姓名（学号）"格式引用，如"软件2301-陈小鹏（120231080315）"。
