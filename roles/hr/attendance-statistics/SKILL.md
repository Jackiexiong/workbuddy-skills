# 考勤统计（开发中）

> ⚠️ 本技能为占位文件，待开发实现。

## 规划功能

- 考勤数据导入与清洗（打卡机导出 Excel / CSV）
- 出勤率、迟到、早退、缺勤自动判定
- 加班时长统计与调休折算
- 请假类型汇总（年假 / 病假 / 事假 / 调休）
- 异常打卡标记与待确认清单
- 部门 / 个人维度的月度汇总报表（Excel）

## 使用场景

- 月度考勤结算与薪资核算依据
- 部门出勤情况分析
- 个人考勤异常追踪与提醒
